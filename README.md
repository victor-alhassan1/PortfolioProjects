# COVID-19 Death Rate Analysis Dashboard
### Overview
This project analyzes COVID-19 mortality data across different geographical locations, focuaing on total death counts and death rates (%). 
The goal was to transform raw pandemic data into clear, interactive visual insights for regional comparison.

## Dataset
Source: Our World in Data
Includes country-level data on confirmed cases, total deaths, and population.

## Tech Stack
- Excel - Data cleaning and preparation
- MySQL - Data querying and calculation of mortality metrics
- Tableau - Interactive dashboard design
- Github - Documentation

### Key Metric
Death Rate (%) = (Total Deaths / Total Confirmed Cases) x 100

## Highlights
- Geographic comparison of total deaths
- Mortality rate analysis across regions
- Interactive filters for deeper exploration

## Project Value
This project demonstrates an end-to-end analytics workflow and reflects my interest in leveraging healthcare data for evidence-based insights.
